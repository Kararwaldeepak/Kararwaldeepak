C:\THz> Δt = mΔL / c
C:\THz> L_scan = (N − 1)ΔL
C:\THz> T_span = (N − 1)Δt
C:\THz> Δf = 1 / (NΔt)
C:\THz> f_Nyquist = 1 / (2Δt)
C:\THz> N_f = floor(N/2) + 1
C:\THz> τ_FWHM = t₂ − t₁
